## Installation

*Installation*
```sh
npm i @vanillawc/wc-code
```

*Import from NPM*
```html
<script type="module" src="node_modules/@vanillawc/wc-code/index.js"></script>
```

*Import from CDN*
```html
<script type="module" src="https://cdn.jsdelivr.net/gh/vanillawc/wc-code/index.js"></script>
```
